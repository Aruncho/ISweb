¿Que es InvenSoftware Administrativo?

¡Aqui tienes las respuestas!

Modulo de Inventario

Módulo que permite controlar y hacer seguimiento a las existencias, precios y costos de productos de la empresa.

Una de la opciones más importantes que tiene este módulo es la del Registro de Productos, donde definimos las características y propiedades de cada item, es decir Código, Descripción, Tipo de Producto, Línea de Producto, SubLínea de Producto, Marca, Presentación, etc.
Los movimientos del Almacén se controlan a través de documentos como: Ingresos o Egresos por Ajustes, etc. Se pueden realizar transferencias entre almacenes generando automáticamente la Nota de Ingreso y Nota de Salida correspondientes.
Este módulo se encuentra integrado con el módulo de Compras, para ello permite generar una Nota de Ingreso en base a las Ordenes de Compra registradas, con lo cual obtenemos un control sobre lo que se pide y lo que se recibe. También nos permite valorizar el almacén automáticamente. Generando un Informe de Listado de Existencias por Costos.

Este módulo nos proporciona informes tales como: Existencia Actual, Kardex por Rango de Fecha, Kardex Valorizado, etc. Estos informes los podemos obtener en línea y en cualquier momento que la Gerencia lo requiera.

Modulo de Compras

Este módulo le permite obtener el control de la parte operativa referente a la compras ó gastos de su Empresa.
Con la finalidad de proporcionar información oportuna a la Gerencia. Informes de Compras y Gastos.

Toda operación de compra se inicia con una Orden de Compra, donde especificamos las Condiciones de pago, Productos a Comprar, Fecha y Lugar de Entrega, etc. que es remitida al Proveedor. El Proveedor envía los productos y quien lo recepciona es área de Almacén donde registra la Nota de Ingreso Por Compra.
Registrado el Documento se genera automáticamente la Cuenta por Pagar del Proveedor, es decir, que la Gerencia puede visualizar la Deuda por Pagar en línea. También podemos visualizar el Registro de Compras, así como las Estadísticas de Compras inmediatamente.

Modulo de Ventas

El módulo de Ventas utiliza dos esquemas de venta: uno es el de Venta con Despacho, orientada a empresas cuyas operaciones se realizan a través de despachos totales o parciales en base a una orden de pedido. El otro esquema es el de Venta Mostrador, orientado a empresas con operaciones donde el despacho al cliente es inmediato.

Para las Ventas con Despacho el módulo permite registrar una Factura y finalmente generar generar una Guía de Despacho. La Factura genera automáticamente un movimiento en Almacén, afectando al Stock.
Para el caso de Venta Mostrador el sistema genera un pre-documento (Pedido) y luego el Documento de Venta (Factura), internamente se genera la Nota de Salida que afecta al módulo de Almacén descargando el stock. Automaticamente imprime una Comanda en Almacen para la Preparación de los Productos para su respectiva entrega inmediata al cliente.
Una vez generado el Documento de Venta el módulo automáticamente afecta la Cuenta por Cobrar del Cliente, de tal manera que podemos obtener su estado de cuenta en línea. Otra información a obtener en línea son estadísticas, tales como: Ventas por Cliente, Ventas por Vendedor, Ventas por Producto, etc. Además podemos obtener el Registro de Ventas aplicando cualquier fltro: (Fechas Desde-Hasta, Vendedor, Clientes, Zona, Canal, etc).

Integrado para impresoras Fiscales de Venezuela.
Integrado con la Facturación Electronica Ecuador.

Modulo de Despacho

Con el Módulo de Despacho podrá hacer seguimiento de la mercancía si esta fue entregada con exito o no. Validar las entregas, reasignar nueva transportista para el envío en caso de no haberse entregado. Y generar informes de los despachos del dia, para saber cuales fueron efectivos, cuales no se han realizado, y medir la eficiencia del transportista.

Modulo de Banco

Este módulo nos permite llevar un control de las operaciones financieras de la empresa, con la finalidad de obtener información oportuna para la Gerencia.

Este módulo nos permite registrar los movimiento Bancarios, de tal manera que podamos obtener inmediatamente nuestros estados de cuenta. Hay operaciones que afectan al Libro Banco, tales como el Pago a Proveedores si es con cheque o transferencias.
Tambien nos permite conciliar nuestras operaciones Bancarias con el estado de Cuenta del Banco.

Para cada movimiento Bancario, este módulo le asigna un concepto a dicho movimiento, de tal manera que podamos obtener un Flujo de Caja Realizado Diario. También podemos obtener un Flujo Caja Mensual Realizado, donde nos muestra el movimiento Bancarios mes a mes con sus saldos respectivos.
